# Allocoint – Budget and Expense Tracker (Tkinter)

## Overview

Allocoint is a desktop-based budget and expense tracking application developed using Python and Tkinter. It was created as a requirement for an Advanced Computer Programming course. The system allows users to allocate a total budget into different categories and track expenses in real time through a graphical user interface.


## Features

* Set total budget or funds
* Create and manage budget categories
* Allocate funds per category
* Track expenses per category
* Real-time balance updates
* Remaining funds monitoring
* Input validation and error handling
* Tab-based user interface using Tkinter Notebook


## How It Works

1. The user inputs a total budget.
2. The budget is distributed into different categories.
3. Each category maintains its own budget and remaining balance.
4. The user records expenses under each category.
5. The system automatically updates all balances and remaining funds.


## Technologies Used

* Python 3
* Tkinter (Graphical User Interface)
* Object-Oriented Programming (OOP)
* Modular programming structure
